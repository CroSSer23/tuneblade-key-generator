# 🇷🇺 Русский

## 📋 Описание

TuneBlade Key Generator - это веб-приложение для генерации ключей активации программы TuneBlade. Приложение работает полностью на стороне клиента, не передавая никаких данных на сервер, что обеспечивает приватность и безопасность.

### Особенности:

- ✅ Мгновенная генерация ключей активации
- 🔒 Безопасно: весь код выполняется только на вашем устройстве
- 🌍 Поддержка трех языков: Русский, English, 中文
- 📱 Адаптивный дизайн для мобильных и десктопных устройств
- 📋 Пошаговые инструкции по активации
- 📋 Копирование ключа в буфер обмена одним кликом

## 🚀 Как пользоваться

### Шаг 1: Генерация ключа

1. Откройте [генератор ключей](https://crosser23.github.io/tuneblade-key-generator/)
2. Введите свой email-адрес в поле ввода
3. Нажмите кнопку "Сгенерировать ключ"
4. Скопируйте сгенерированный ключ, нажав кнопку "Копировать"

### Шаг 2: Активация TuneBlade

После генерации ключа следуйте инструкциям, которые появятся под формой:

1. Откройте программу TuneBlade и перейдите в меню "Настройки" (Preferences)
2. Выберите раздел "License" в левой панели
3. В поле "License ID" введите ваш email адрес
4. В поле "License Key" вставьте сгенерированный ключ
5. Нажмите кнопку "Unlock"

## 📝 Как это работает

Генератор использует хеш-алгоритм MD5 для создания ключа на основе вашего email-адреса. Ключ генерируется по специальному алгоритму, совместимому с системой проверки лицензий TuneBlade.

Весь процесс происходит полностью в вашем браузере, без отправки данных на сервер.

# 🇬🇧 English

## 📋 Description

TuneBlade Key Generator is a web application for generating activation keys for the TuneBlade program. The application runs entirely on the client side, without sending any data to the server, ensuring privacy and security.

### Features:

- ✅ Instant activation key generation
- 🔒 Secure: all code is executed only on your device
- 🌍 Support for three languages: Russian, English, Chinese
- 📱 Responsive design for mobile and desktop devices
- 📋 Step-by-step activation instructions
- 📋 One-click key copying to clipboard

## 🚀 How to Use

### Step 1: Generate a Key

1. Open the [key generator](https://crosser23.github.io/tuneblade-key-generator/)
2. Enter your email address in the input field
3. Click the "Generate Key" button
4. Copy the generated key by clicking the "Copy" button

### Step 2: Activate TuneBlade

After generating the key, follow the instructions that appear below the form:

1. Open the TuneBlade program and go to "Preferences"
2. Select the "License" section in the left panel
3. Enter your email address in the "License ID" field
4. Paste the generated key in the "License Key" field
5. Click the "Unlock" button

## 📝 How It Works

The generator uses the MD5 hash algorithm to create a key based on your email address. The key is generated using a special algorithm compatible with the TuneBlade license verification system.

The entire process takes place completely in your browser, without sending data to the server.
